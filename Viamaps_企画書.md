# Viamaps ビジネス企画書

Google マイマップ × tabiori の機能を持つ旅行地図サービス

2026年5月 / Version 2.0

---

## 1. エグゼクティブサマリー

Viamaps は、**Google マイマップ**の地図機能と**tabiori**の旅程管理機能を統合した旅行地図 Web サービスである。

地図上にポイントを作り、**日程（Day 1・2・3…）ごとに整理**し、Mapbox Directions API による**実際の道路ルート**を自動表示。「← →」ボタンで**地図を閉じることなくポイント間をスムーズにナビゲーション**できるコア体験に加え、チェックリスト・費用管理・描画ツールなどを備える。

旅行計画・旅行記録・観光スポット共有のあらゆる用途で使える汎用マップツールとして展開する。

### 1.1 解決する課題

Google マイマップ／tabiori で旅行者が直面する問題：

| サービス | 課題 |
|---------|------|
| Google マイマップ | ポイントAを見る→地図に戻る→繰り返しが苦痛。日程管理が弱い |
| tabiori | 地図機能が限定的。道路ルートや距離が確認しにくい |

### 1.2 ターゲットユーザー

- **旅行計画者**：友人・家族との旅行を日程単位で地図にまとめたい
- **旅行記録者**：行った場所・写真・感想・費用を地図に残したい
- **観光情報共有者**：おすすめスポットをルートつきでビジュアル共有したい

## 2. プロダクト概要

### 2.1 コアバリュー

> 「地図を何度も開き直さなくていい。日程ごとに整理して、← → だけで旅が進む。」

### 2.2 競合サービスとの差別化

| 機能 | Google マイマップ | tabiori | Viamaps |
|------|-----------------|---------|---------|
| ポイント間移動 | **毎回地図を開き直す** | リスト中心 | **← → でアニメーション移動** |
| 日程管理 | 弱い（レイヤー） | **Day 1・2・3…** | **Day 1・2・3… + 地図連動** |
| 道路ルート | あり | なし | **Mapbox Directions API** |
| 距離・所要時間 | あり | なし | **自動表示（日程ヘッダー）** |
| チェックリスト | なし | **あり** | **あり（持ち物・やること）** |
| 費用管理 | なし | **あり** | **あり（ポイントごと）** |
| 描画ツール | あり | なし | **あり（ライン描画）** |
| 埋め込みコード | あり | なし | **あり（iframe）** |
| 共有 | Googleログイン要 | アプリ要 | **ログイン不要リンク** |
| モバイル操作性 | 重い | 専用アプリ | スマホ最適化 Web |

### 2.3 主要機能

#### マップ作成・編集
- マップに名前・説明を付けて作成
- 地図上をクリック、または **場所検索（Mapbox Geocoding）** でポイントを追加
- ポイントにタイトル・説明・画像URL・カテゴリ・時刻・費用を入力
- **日程（Map Days）ごとにポイントを整理**
- ポイントの並び順を変更
- **マーカーカラーを個別設定**（10色）
- **地図スタイル切替**（地図 / 衛星 / 地形）
- **描画ツール**（ライン描画・色設定）

#### 日程管理（tabiori 相当）
- Day 1・Day 2・Day 3… と日程を追加
- 各日程に日付・タイトルを設定
- 日程ごとに色分けされたマーカー・ルート線
- **Mapbox Directions API** による実際の道路ルート + 距離・所要時間

#### チェックリスト（tabiori 相当）
- 持ち物リスト（packing）
- やることリスト（todo）
- チェック ON/OFF・追加・削除

#### ビューアモード（コア差別化機能）
- デスクトップ：左サイドパネル（日程・ポイント一覧）+ 右地図
- 「←」「→」で次のポイントに flyTo アニメーション
- 日程別カラーマーカー・ルート線
- 画像スライダー・ポイント詳細表示

#### 共有・埋め込み
- 公開 ON にすると URL で誰でも閲覧可能
- **iframe 埋め込みコード**生成・コピー
- ログイン不要で閲覧できる

## 4. 技術スタック

| カテゴリ | 技術 |
|---------|------|
| フロントエンド | Next.js 15 (App Router) + React 19 + TypeScript |
| スタイリング | Tailwind CSS v4 |
| 地図ライブラリ | Mapbox GL JS（react-map-gl 経由） |
| 地図 API | Mapbox Geocoding API・Mapbox Directions API |
| バックエンド | Next.js Server Actions |
| データベース | Supabase（PostgreSQL + RLS） |
| 認証 | Supabase Auth（メール/パスワード + Google OAuth） |
| ホスティング | Vercel |
| エラー監視 | Sentry |
