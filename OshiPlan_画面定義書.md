2026年5月 / Version 2.0

## 0. 画面一覧・遷移概要

### ナビゲーション構造

```
[全ユーザー（未ログイン含む）]
  ├── P-00 トップページ（/）
  ├── P-01 ログイン（/auth/login）
  ├── P-02 新規登録（/auth/register）
  ├── P-10 会場一覧（/venues）
  ├── P-11 会場別LP（/venue/[slug]）  ← SSG・SEO最重要
  ├── P-30〜34 プラン作成フロー（ゲスト可）
  └── P-25 共有プラン閲覧（/shared/[token]）

[ログイン必要]
  ├── P-20 プラン一覧（/plans）
  ├── P-21 プラン詳細（/plans/[id]）
  ├── P-22 プラン編集（/plans/[id]/edit）
  ├── P-40 推し一覧（/artists）
  ├── P-41 推し登録（/artists/new）
  ├── P-42 推し編集（/artists/[id]/edit）
  └── P-50 設定（/settings）
```

## 8. 共通仕様

### エラー・空状態

| パターン | 表示 |
|---------|------|
| ネットワーク切断 | トースト: 「インターネット接続を確認してください」 |
| API エラー（5xx） | インライン: 「エラーが発生しました。しばらくしてから再試行してください」 |
| AI生成レート超過（未ログイン） | モーダル: 「本日の生成上限（3回）に達しました。ログインすると10回まで使えます」＋ログインCTA |
| AI生成レート超過（ログイン済み） | モーダル: 「本日の生成上限（10回）に達しました。明日また使えます」 |
| データなし（空状態） | イラスト＋説明テキスト＋CTAボタン |

**変更点**: 旧仕様の「Premiumアップグレード促進モーダル」を「ログイン促進モーダル」に変更。

### OGP設定（SNSシェア）

プランの共有URLには専用のOGP画像を生成（Next.js の ImageResponse を使用）。

```
OGP画像例:
┌─────────────────────────┐
│ 🎵 OshiPlan             │
│ ○○ ARENA TOUR 2026     │
│ 東京ドーム・1泊2日       │
│ 概算 ¥38,000            │
│ 名古屋発                │
└─────────────────────────┘
```

X/LINE でシェアした際に「見栄えするカード」が表示され、クリック率向上・新規流入を促進する。
