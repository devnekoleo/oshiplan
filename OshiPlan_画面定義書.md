推し活遠征プランナーアプリ

2026年5月 / Version 1.0 (MVP)

## 0. 画面一覧・遷移概要

### ナビゲーション構造

```
[未認証]
  ├── S-01 ログイン
  └── S-02 新規登録

[認証済み] BottomTab
  ├── Tab: ホーム        → S-10 ホーム
  ├── Tab: プラン        → S-20 プラン一覧
  ├── Tab: ＋（作成）    → S-30 プラン作成 Step1
  ├── Tab: 推し          → S-40 推し一覧
  └── Tab: 設定          → S-50 設定

[認証不要]
```

## 1. 認証系

### S-01 ログイン

| 項目 | 内容 |
|------|------|
| 目的 | ユーザーがアカウントにログインする |
| 認証 | 不要 |

**UI要素**

```
┌─────────────────────────────┐
│  🎵 OshiPlan                │
│     推し活遠征プランナー       │
│                             │
│  [メールアドレス入力]         │
│  [パスワード入力]             │
│                             │
│  [─── ログイン ───]          │
│                             │
│  ─────── または ───────      │
│                             │
│  [🍎 Appleでログイン]         │
│  [G  Googleでログイン]        │
│                             │
│  パスワードを忘れた方はこちら   │
│  アカウントをお持ちでない方へ   │
└─────────────────────────────┘
```

| UI要素 | 種別 | バリデーション |
|--------|------|--------------|
| メールアドレス | TextInput | email形式 |
| パスワード | TextInput（秘匿） | 8文字以上 |
| ログインボタン | Button | — |
| Appleでログイン | Button | — |
| Googleでログイン | Button | — |
| パスワードリセットリンク | Link | → S-03 |
| 新規登録リンク | Link | → S-02 |

**アクション**
- ログイン成功 → S-10 ホーム
- ソーシャルログイン成功 → S-10 ホーム（初回は S-51 プロフィール設定へ）
- エラー → インライン表示（「メールまたはパスワードが違います」）

## 8. 共通仕様

### エラー・空状態

| パターン | 表示 |
|---------|------|
| ネットワーク切断 | トースト：「インターネット接続を確認してください」 |
| API エラー（5xx） | インライン：「エラーが発生しました。しばらくしてから再試行してください」 |
| データなし（空状態） | イラスト＋説明テキスト＋アクションボタン（画面別） |
| Free枠超過 | モーダル：Premiumアップグレード訴求 |

### ローディング

| パターン | 表示 |
|---------|------|
| 画面初期表示 | スケルトンUI（カードの形でグレーアニメーション） |
| ボタン押下後 | ボタン内スピナー＋非活性化 |
| AI生成中 | S-33 専用ローディング画面 |

### 共通ナビゲーション（BottomTabBar）

| タブ | アイコン | 遷移先 |
|------|---------|--------|
| ホーム | 🏠 | S-10 |
| プラン | 📋 | S-20 |
| ＋（作成） | ➕（中央・強調） | S-30 |
| 推し | ⭐ | S-40 |
| 設定 | ⚙️ | S-50 |
