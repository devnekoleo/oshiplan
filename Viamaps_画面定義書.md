### 3.2 主要コンポーネント

#### 3.2.1 DraftBanner（下書きバナー）
- 表示条件：`draftStore.draft !== null`
- 内容：「未保存の地点があります『〇〇』 [破棄] [保存]」
- 色：amber-100 背景、上から sticky
- 「保存」クリック → PointPanel を開く（タイトル等を入力）→ 保存完了で消える
- 「破棄」クリック → 確認モーダル → draft を null に

#### 3.2.2 ModeToggle（モード切替）
```
┌─────────────────────────────┐
│ [👁 閲覧] [📍 追加] [✏️ 描画] [📏 計測] │
└─────────────────────────────┘
```
- 地図右上に floating
- アクティブモードは bg-blue-600 text-white
- ホットキー: V / A / D / M

#### 3.2.3 PlacesAutocomplete（場所検索）
- 地図上部 floating
- 入力 → 300ms debounce → `/api/places/autocomplete`
- 候補クリック → そのまま下書きピン配置 + 地図 panTo

#### 3.2.4 PointSidebar（左サイドバー）
- Day ごとに折りたたみアコーディオン
- Day ヘッダー：色チップ + タイトル + 距離 + 時間 + 費用 + ▼
- ポイント行：マーカー色チップ + アイコン + タイトル + 時刻
  - クリック → flyTo + 選択
  - ドラッグ → 並び替え（同 Day 内 / Day 間）
  - 行末 […] メニュー：編集 / 削除 / Day 変更

#### 3.2.5 PointPanel（右スライドオーバー）
- 表示条件：`editorStore.editingPointId !== null` または draftを編集中
- 幅：480px（モバイルでは全幅ボトムシート）
- フィールド：
  - 📷 画像（プレビュー / アップロード / Places 写真選択）
  - タイトル *
  - カテゴリ（spot / hotel / restaurant / station / sightseeing / shopping / activity / transport / other）
  - 説明
  - 時刻（開始 / 終了）
  - 費用（円）
  - マーカー色（10色ピッカー）
  - Day 割り当て（プルダウン）
  - 📍 Place 情報（Place ID あり時のみ）：評価・電話・営業時間・公式サイトリンク
- フッター：[削除] [保存]
- ✕ で閉じても下書きは保持

#### 3.2.6 GoogleMapCanvas
- `@vis.gl/react-google-maps` ベース
- Advanced Markers（Day 色 + カテゴリアイコン）
- 下書きピン（緑＋点滅 CSS animation）
- Directions Polyline（Day 色）
- Drawing Layer（mode === 'draw' 時のみ）
- InfoWindow（ホバー時のみ表示、クリックで PointPanel）

#### 3.2.7 ナビゲーションボタン
- 地図下中央 floating「← 前へ ｜ 1 / 12 ｜ 次へ →」
- 現在の選択ポイントから順次 flyTo

### 3.3 モバイルレイアウト

```
┌──────────────────────────┐
│ Header                   │
├──────────────────────────┤
│ ⓘ 未保存（東京タワー）   │
├──────────────────────────┤
│                          │
│      Google Map          │
│       (全画面)            │
│                          │
│  ◀  ▶  ←ナビ            │
│                          │
│  [👁][📍][✏️][📏] ← Mode │
├──────────────────────────┤
│ ▲ ボトムシート（Day一覧）│
│   ドラッグで展開          │
└──────────────────────────┘
```

PointPanel はモバイルでは全画面ボトムシート（80vh）として表示。

### 3.4 状態と振る舞いマトリクス

| 操作 | mode=view | mode=add | draft あり | editing あり |
|------|----------|----------|----------|------------|
| 地図クリック | 何もしない or 空白選択解除 | 下書きピン配置/移動 | 下書きピン移動 | 下書きピン移動 or 配置 |
| マーカークリック | 選択 + InfoWindow | 同左 | 同左 | 同左 |
| マーカー Dbl-click | PointPanel オープン | 同左 | 同左 | 別ポイントに切替 |
| サイドバー Day 切替 | 即切替 | 即切替 | 即切替 | 即切替 |
| パネル ✕ | - | - | draft 維持 | editing 解除 (draft 維持) |
| Esc | サイドバー閉じる | mode=view へ | 破棄確認 | パネル閉じる |

## 9. SC-031 iframe ビューア

最小化レイアウト：
- ヘッダーなし
- 地図全画面 + 簡易 Day セレクタ（タブ）
- ← → ナビボタン
- 右下「Viamaps」リンク（クリックで元マップへ）

クエリパラメータ：
- `?day=2` 初期日程
- `?mode=light|dark`
- `?hideRoute=true` ルート非表示

## 12. アクセシビリティ要件

- すべてのインタラクティブ要素にフォーカスリング
- ModeToggle はラジオグループ ARIA
- DraftBanner は `role="alert"` `aria-live="polite"`
- PointPanel は `role="dialog"` `aria-modal="false"`（モーダルにしない）
- マーカーには `aria-label`（タイトル + 日程）
