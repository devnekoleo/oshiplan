# Viamaps 画面定義書

Google マイマップ × tabiori の機能を持つ旅行地図サービス

2026年5月 / Version 2.0

---

## 1. 画面一覧

| 画面名 | URL | 認証 | 主な機能 |
|--------|-----|------|---------|
| ランディングページ | `/` | 不要 | サービス紹介・マップ作成CTA |
| ログイン | `/auth/login` | 不要 | メール/パスワード + Google OAuth |
| 新規登録 | `/auth/register` | 不要 | Google OAuth 優先 + メール登録 |
| パスワードリセット | `/auth/reset-password` | 不要 | リセットメール送信 |
| マイマップ一覧 | `/maps` | 要 | カードグリッド・マップ管理 |
| マップ作成 | `/maps/new` | 要 | 新規マップ作成フォーム |
| マップエディタ | `/maps/[id]` | 要（オーナー） | 日程・ポイント・描画ツール |
| マップ設定 | `/maps/[id]/settings` | 要（オーナー） | タイトル・公開設定・埋め込みコード |
| チェックリスト | `/maps/[id]/checklist` | 要（オーナー） | 持ち物・やること管理 |
| ビューアモード | `/maps/[id]/view` | 任意 | スムーズナビゲーション |
| 共有マップ | `/shared/[token]` | 不要 | 公開マップ閲覧（読み取り専用） |
| 設定 | `/settings` | 要 | アカウント設定 |

---

## 2. 各画面の詳細

### 2.1 ランディングページ（`/`）

**構成**:
```
[ヘッダー] 📍 Viamaps | マップを作る | マイマップ | ログイン/登録

[Hero]
  📍（大きいアイコン）
  旅を、地図で語ろう。
  「← → で地図を開かずにスムーズに移動」
  [🗺️ マップを作る（無料）]  [マイマップを見る（ログイン済みの場合）]

[特徴セクション]
  ⚡ 地図を開き直さない  📸 写真＋説明をセットで  🔗 リンクで簡単共有

[CTAセクション]
  「ログイン不要で今すぐ試せます」  [無料でマップを作る →]
```

### 2.6 マップエディタ（`/maps/[id]`） ⭐ メイン編集画面

**ツールバー**:
```
← マイマップ | マップ名 | [公開中/非公開] | [📋 チェックリスト] [▶ ビューア] [設定]
```

**レイアウト（デスクトップ）**:
```

│  左サイドバー（w-72）           │  Mapbox 地図（メイン）                    │
│                                │                                          │
│  [🔍 場所を検索...            ] │  ① ② ③  番号付きカラーマーカー            │
│                                │                                          │
│  [🗺️地図][🛰️衛星][⛰️地形]    │  ───── Day1ルート（Directions API）─────   │
│                                │  - - - - Day2ルート - - - - - - - -       │
│  [＋ 日程を追加] [✏️ ライン]   │                                          │
│                                │  右上：ナビゲーションコントロール          │
│  ▼ ● Day 1  2026/05/23        │                                          │
│    🚗 12.3km · 45分            │                                          │
│    [1] 東京タワー    9:00      │  [ホバー時ポップアップ]                   │
│    [2] 浅草寺       11:00     │  東京タワー                               │
│    [＋ スポットを追加]         │  🕐 9:00〜11:00  💴 ¥1,800               │
│                                │  東京の有名な観光スポット                 │
│  ▶ ● Day 2 （折りたたみ中）   │                                          │
│                                │                                          │
│  ── 日程なし ──                │                                          │
│    [1] スカイツリー            │  [右オーバーレイパネル（ポイント選択時）]  │
│                                │  ✕ ポイントを追加 / 編集                 │
```

**描画モード（ライン描画）**:
```
[サイドバー上部に描画パネルが出現]
  地図をクリックしてラインを描画。ダブルクリックで完了。
  ライン名（任意）[入力]
  [●青][●赤][●緑][●黄][●紫][●ピンク]  カラー選択
  [完了 (3点)]  [キャンセル]
```

**ポイント追加/編集パネル（右オーバーレイ）**:
```
✕  ポイントを追加 / ポイントを編集

タイトル * [入力]

カテゴリ  [📍スポット] [🍽グルメ] [🏨ホテル] [🚗移動]

マーカーカラー  [グラデ=日程カラー] [●青][●緑]...(10色)
                ↑ グラデ選択=日程カラーに従う

日程  [Day 1 — 東京観光 (2026/05/23) ▼]

開始時刻 [09:00]  終了時刻 [11:00]

費用  [1800] 円

説明 [テキストエリア]

画像
  [追加済み画像サムネイル] [✕]
  URL [入力]  キャプション [入力]
  [＋ 画像を追加]

  [追加する / 保存する]
  [このポイントを削除]（編集時のみ）
```

### 2.10 共有マップ（`/shared/[token]`）

ビューアモードと同じ UI。「✏️ 編集」リンクは表示しない。share_token で公開マップのみアクセス可。

## 5. レスポンシブ対応

| 要素 | モバイル | デスクトップ |
|------|---------|------------|
| エディタ サイドバー | 非表示 | 左側に表示（w-72） |
| ポイントパネル | 画面全体 | 右側オーバーレイ（w-80） |
| ビューアレイアウト | 地図上 + 情報下 | 左パネル + 右地図 |
| マップ一覧 | 1カラム | 2カラム |
| ビューア左パネル | 非表示 | 表示（w-72） |
| チェックリスト | 1カラム | 最大幅 max-w-2xl |
